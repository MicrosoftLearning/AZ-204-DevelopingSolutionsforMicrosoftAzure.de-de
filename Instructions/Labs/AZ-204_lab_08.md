    | Einstellung | Aktion |
    | -- | -- |

    | Dropdownliste **Imagequelle** | Wählen Sie **Docker Hub**. |
    | Dropdownliste **Zugriffstyp** | Wählen Sie **Öffentlich (10.0.0.0/24)** . |
